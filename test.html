Display Simple HTML Without MS Controls

Speed Bug FIXED!
This program will display very simple html files without using a MS Web Browser control.
Some of the features include:

Bold Text
Italic Text
underline text
Font Sizes
Font Faces
Font Colors
Hyperlinks

Suggested Modifications:
Text Alignment
Images

Please bear in mind that this is by no means the Alpha and Omega of web browsers.
This just displays local (on your hardrive) html files.

So why build this instead of using an MS Web Browser?
Simple, some programs I write need to display basic HTML files but I do not want or need all the features of the MS Web Browser.
